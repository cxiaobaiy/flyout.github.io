聊天室成员 ​

功能说明 ​

客服查询群成员

接口调用说明 ​

接口地址:/user/service/groupUser post application/json

请求参数 ​

Header参数属性描述
Authorization必填客服登录接口返回的userSig参数
Body参数类型属性描述
pageint必填分页数据:分页下标，从1开始
sizeint必填分页数据:分页大小
groupIdint必填聊天室id(会话列表type为GROUP的会话，字段recevId就是groupId)
searchNameString选填搜索字段(根据uuid和昵称搜索)
levelString选填用户等级
shutupint选填0未禁言 1禁言

请求示例 ​

json
{
  "page":1,
  "size":1,
  "groupId":1
}
