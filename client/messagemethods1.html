1.创建文本消息 createTextMessage ​

tips:请在登录后使用

参数：

keystatus字段描述
to:string,number用户的userId 或 群聊ID
conversationId:string会话Id,取自会话列表
conversationType:string会话列表 可选（C2C 单聊，GROUP 群聊）
text:string发送的文本
needReadReceipt:Boolean可选是否需要已读回执，默认为false,即不需要已读回执
atUserList:Array可选@ 成员名单，单聊无效
hasRiskContent:Boolean可选消息进行风险标识

返回值：Promise<Object>

eg：

ecmascript

import {useFxIm} from 'fx-im-web'

const fxIm = useFxIm()
fxIm.createTextMessage({
    to:1,
    text:'我是测试消息',
    conversationType:'C2C',// 可选GROUP发送群聊消息
    conversationId: 'C2CtestUser' 
}).then(res=>{
    console.log(res,'创建的文本消息')
})

推荐使用 createMessage 创建文本后直接上屏渲染 减少等待时间

2.发送消息 sendMessage ​

tips:请在登录后使用

参数：

message 由createxxMessage 返回的对象

返回值：Promise<Object>

eg：

ecmascript

import {useFxIm} from 'fx-im-web'

const fxIm = useFxIm()
fxIm.createTextMessage({
    to:1,
    text:'我是测试消息',
    conversationType:'C2C',// 可选GROUP发送群聊消息
    conversationId: 'C2CtestUser' 
}).then(res=>{
    fxIm.sendMessage(res.data).then(res=>{
        //可以在这里处理消息状态变更等操作
    })
})

4.删除消息 deleteMessage ​

tips:请在登录后使用

参数：

message 由createxxMessage 返回的对象 完整的消息对象即可

返回值：Promise<Object>

eg：

ecmascript

import {useFxIm} from 'fx-im-web'

const fxIm = useFxIm()
fxIm.deleteMessage(
    {
        "id": "1818143398549393408",
        "conversationType": "GROUP",
        "conversationId": "GROUPtest",
        "sendUserInfo": {
            "nick": "nickName",
            "id": 1,
            "avatar": "https://ts1.cn.mm.bing.net/th?id=OIP-C.Zte3ljd4g6kqrWWyg-8fhAHaEo&w=316&h=197&c=8&rs=1&qlt=90&o=6&pid=3.1&rm=2"
        },
        "time": 1722314141287,
        "messageType": "TEXT_MESSAGE",
        "payload": {
            "text": "我是测试消息"
        },
        "flow": "out",
        "sendId": 1,
        "recevId": 1,
        "status": "success",
        "isRevoked": false,
        "nick": "nickName",
        "avatar": "https://ts1.cn.mm.bing.net/th?id=OIP-C.Zte3ljd4g6kqrWWyg-8fhAHaEo&w=316&h=197&c=8&rs=1&qlt=90&o=6&pid=3.1&rm=2",
        "isPeerRead": false,
        "isDeleted": false,
        "isModified": false,
        "needReadReceipt": false,
        "hasRiskContent": false,
        "atUserList": []
    }
).then(res=>{
    console.log('删除消息成功')
})

注意：删除分为管理员操作和非管理员操作，管理员操作为物理删除，管理员删除后，在消息记录中将无法再次获取该消息，且管理员操作将会影响群聊所有用户 非管理员和单聊操作删除，将仅影响用户本身，且为逻辑删除 即不会通知对方或群聊成员，仅自己不可见

5.群组设置禁言 setGroupSilence ​

tips:请在登录后使用

参数：

params { groupId: 群组id , userId： 需要禁言的用户ID}

返回值：Promise<Object>

eg：

ecmascript

import {useFxIm} from 'fx-im-web'

const fxIm = useFxIm()
fxIm.setGroupSilence({
    userId:1,
    groupId:'GroupTestid1'
}).then(res=>{
    console.log('禁言成功')
})
